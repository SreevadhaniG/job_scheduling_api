# Job Scheduling API
Job Scheduling API is a simple Python-based service that exposes HTTP endpoints to schedule, manage, and execute background jobs.

## Features
- Create and register jobs with input parameters.
- Schedule jobs for immediate or delayed execution.
- Manage job lifecycle (list, check status, cancel).
- Built with Python and minimal dependencies for easy deployment.

## Tech Stack
- Python 3.x.
- Dependencies listed in `requirements.txt` (install with `pip install -r requirements.txt`).
